How to add a new post

  1. Complete the fields below
  2. Click “Copy to clipboard” (button at the bottom of this page)
  3. Click “Create post” (button at the bottom of this page) – a new tab will open
  4. Paste into the “Edit new file” box (center of the new page)
  5. Name the file in the format 2021-12-31-nameofsong-nameofartist.md (text box at the top of the new page).
    File names must have a date at the start in the format YYYY-MM-DD- or the post won't show up. The date must be today or earlier.
    File names must end in .md or the post will not format correctly.
  6. Click “Commit new file” (button at the bottom of the new page)
